# 更新日志

## v1.2.1

修复：

- 修复 AstrBot 调用 handler 时 `self` 为 None 导致的 `_parse_command` 崩溃
- 兼容 handler 多参数签名；`/画像` 前缀与 command 注册双入口
- 微信场景隐藏 QQ 号（资料 chips / 页脚 / LLM 提示）
- 加入基于 @/引用回复 的关系网统计与【关系网】正文区块

## v1.2.0

新功能：

- 画像结果默认渲染为磨砂可爱风图片卡片（本地统计 + LLM 结构化正文）
- 支持微信群（实时消息缓存采集）；QQ 继续支持历史回溯
- 内置中文字体，降低跨平台乱码风险
- 生成画像时保存 last_stats 等元数据

改进：

- 卡片发送后约 30 秒自动清理临时 PNG
- 命令收敛为单一 `画像`（移除正/负/查看）
- 提示词安全替换 nickname，避免自定义 `{}` 触发 format 异常
- inject_prompt 可注入画像正文（截断）
